Because a hash table has a more complex structure than the data structures we've examined up to this point, we will look at the implementation details in stages. First, we will describe a simple implementation using a fixed table size. This implementation will provide only the most basic methods for adding, removing, and finding keys and values. We will then consider rehashing, or increasing the size of the table as necessary to improve performance. We will then show how to implement enumerators for iterating through the keys and values. Finally, we will show how to implement indexers.
